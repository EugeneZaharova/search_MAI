Список 2020 год в спорте описывает спортивные события 2020 года.

Ожидаемые события
Январь
26 декабря — 5 января — Чемпионат мира по хоккею среди молодёжных команд ( Острава, Тршинец; Чехия).
10—26 января — Чемпионат Европы по гандболу среди мужчин ( Стокгольм, Мальме, Гётеборг Швеция; Вена и Грац Австрия; Тронхейм Норвегия).
10—19 января — III Зимние юношеские Олимпийские игры ( Лозанна, Швейцария).
20—26 января — чемпионат Европы по фигурному катанию ( Грац, Австрия).

Февраль
3—9 февраля — чемпионат четырёх континентов по фигурному катанию ( Сеул, Южная Корея).
12—23 февраля — чемпионат мира по биатлону ( Разун-Антерсельва, Италия).

Март
2—8 марта — чемпионат мира по фигурному катанию среди юниоров ( Таллин; Эстония).
23—29 марта — чемпионат мира по фигурному катанию ( Монреаль; Канада).
28 марта—8 апреля — чемпионат мира по хоккею с мячом ( Иркутск, Шелехов; Россия).

Май
1—17 мая — чемпионат мира по хоккею с шайбой ( Цюрих и Лозанна, Швейцария).

Июнь
12 июня—12 июля — Чемпионат Европы по футболу 2020 ( Европа)
12 июня—12 июля — Кубок Америки по футболу 2020 (Южная Америка)

Июль
24 июля — 9 августа — XXXII летние Олимпийские игры ( Токио, Япония).
июль — Чемпионат Европы по футболу (юноши до 19 лет) ( Северная Ирландия)

Август
25 августа — 6 сентября — XVI летние Паралимпийские игры ( Токио, Япония).

Декабрь
10 — 13 декабря — Финал Гран-при по фигурному катанию ( Пекин; КНР).

Без точных дат
Шахматная олимпиада ( Ханты-Мансийск, Россия).
Чемпионат мира по настольному теннису ( Пусан, Южная Корея).
